About Srabeling Apps

Srabeling Apps is a web application built using Flask, specifically designed to assist researchers in extracting data from the Google Play Store. By leveraging the Google Play Scraper library, this application allows users to easily gather the necessary information from various apps available on the platform. Srabeling Apps provides convenience and efficiency for researchers who need to access app data for further analysis, making it a highly valuable tool for mobile application research.

In addition to the data extraction feature, Srabeling Apps also offers an advanced sentiment classification capability. Using a model built with the Support Vector Machine (SVM) algorithm, this application can categorize user reviews into two classes: positive and negative. The model has been tested and achieved an accuracy of 81%, ensuring reliability in sentiment analysis. The combination of data extraction and sentiment labeling makes Srabeling Apps a comprehensive solution for researchers who want to explore and gain deeper insights into the dynamics of apps on the Google Play Store.

How to Use Srabeling Apps

Using Srabeling Apps is simple and straightforward! Just follow these easy steps:

  1. Start by entering the Application ID of the app you wish to scrape in the provided input field. This ID can typically be found in the URL of the app page on the Google Play Store. You can see an example below:
  2. Specify the number of comments you want to scrape by entering a value in the limit field. The application will collect the specified number of comments.
  3. Click the "Scrape" button to initiate the data scraping process. Wait a moment while the application gathers the data for you.
  4. Once the scraping is complete, you will be redirected to a new page displaying the results:
    • You will have the option to download the scraped data directly by clicking on the "Download Hasil Scraping" button.
    • If you wish, you can also classify the comments using the SVM model by clicking the "Klasifikasikan dengan SVM" button. This will categorize the comments into positive and negative.
    • After classification, you can download the classified results by clicking the "Download Hasil Klasifikasi" button.
  5. Review the results to gain insights from the user feedback and use this information for your research or analysis.
